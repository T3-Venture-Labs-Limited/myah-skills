# Honcho Memory Inspector

This skill lets you inspect and understand what Myah remembers about you through the Honcho memory system.

## What You Can Do

- **View memory peers** — See what topics and concepts Myah has learned about you
- **Review conclusions** — Understand the insights Myah has drawn from your conversations
- **Manage memory** — Request deletion or updates to specific memories
- **Export insights** — Get a summary of your memory profile

## Commands

**"What do you know about me?"**
Shows a summary of your memory profile, including key topics and insights.

**"Show my memory peers"**
Lists all the peer categories in your Honcho memory store.

**"What conclusions have you drawn?"**
Displays recent conclusions and insights from your conversations.

**"Delete memory about [topic]"**
Removes specific memories from your store (requires confirmation).

## Privacy Notes

- Your memory data is private and encrypted
- You have full control over what is remembered
- Memories improve Myah's ability to help you
- You can request a complete memory export or deletion at any time

## Example Conversation

**User:** "What do you remember about my work style?"
**Myah:** "Based on our conversations, I've learned that you prefer detailed morning briefings, you work best with structured task lists, and you often need help prioritizing competing deadlines. You also prefer concise email drafts over verbose ones."

**User:** "Delete everything about my personal preferences"
**Myah:** "I'll remove all memory entries related to your personal preferences. This will take effect immediately. Are you sure you want to proceed?"
